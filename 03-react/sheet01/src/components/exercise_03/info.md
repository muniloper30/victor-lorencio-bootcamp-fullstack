## Corregir JSX con errores 
**Se debe entregar componente corregido**

_Lista de errores detectados_

- Componente inicializado en minúsculas, los componentes en React *deben empezar en mayúsculas*
- Faltan los fragmentos _<>_ ya que react exige que solo se devuelva un único elemento padre
- Etiquetas de cierre p faltante y etiqueta img no cerrada, en react _todos las etiquetas deben cerrarse_
- export default sin incluir al final para poder renderizar el componente finalmente
