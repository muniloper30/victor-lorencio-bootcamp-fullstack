## Ejercicio 9. Proyecto corto: panel informativo del bootcamp

**Desarrolla un pequeño panel informativo con contenido relacionado con el bootcamp.**

La interfaz debe incluir, como mínimo:

- Una cabecera principal;
- Un bloque con información general;
- Un bloque con normas o recomendaciones;
- Un bloque con tecnologías del curso;
- Un pie final

_Requisitos_
- Debe estar dividido en varios componentes con nombres significativos.
- App debe actuar como ensamblador principal de toda la interfaz.
- El resultado debe reflejar que entiendes cómo se organiza una interfaz por componentes
