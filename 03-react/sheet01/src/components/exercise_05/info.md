## Ejercicio 5. Mini pantalla de presentación del módulo

**Construye una pequeña pantalla de presentación del módulo de React.**

La interfaz debe incluir, como mínimo:

- Un título principal;
- Una breve introducción;
- Una sección con tres bloques o tarjetas;
- Un cierre final.

_Condiciones_

- Cada bloque o tarjeta debe representarse mediante un componente.
- La pantalla completa debe montarse desde App .
- El resultado debe mostrar claramente una estructura modular.
