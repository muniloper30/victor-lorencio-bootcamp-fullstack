## Ejercicio 9. Proyecto corto: catálogo reutilizable de cursos
### Enunciado
Desarrolla un pequeño catálogo de cursos para la web del bootcamp
_La interfaz debe incluir_
- un título de sección;
- una descripción introductoria;
- una colección de al menos seis cursos;
- cada curso representado por un componente reutilizable

*Cada tarjeta de curso debe mostrar como mínimo:*
- nombre;
- nivel;
- duración;
- modalidad o formato;
- texto de acción

*Restricciones*
- Todo el contenido variable debe llegar por props.
- No repitas componentes casi idénticos.
- El resultado debe reflejar claramente reutilización.
*Criterios de calidad*
- Se valorará especialmente:
- que el diseño del componente sea razonable;
- que las props estén bien nombradas;
- que el JSX sea limpio;
- que el árbol del componente raíz sea fácil de leer
*Criterio extra de calidad profesional*
Antes de terminar, revisa si:
- hay props mal nombradas o ambiguas;
- has dejado información fija dentro del componente que en realidad debería variar;
- podrías mejorar la legibilidad con una mejor estructura
