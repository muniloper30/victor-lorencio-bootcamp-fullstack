## Ejercicio 1 · Formulario básico de inscripción
### Objetivo
* Practicar la estructura general de un formulario y la elección razonable de tipos de campo.
* Dentro de main , construye un formulario para recoger esta información:
 - nombre completo;
 - correo electrónico;
 - teléfono;
 - fecha preferida;
 - mensaje breve.
