## Ejercicio 3 · Formulario con grupos semánticos
### Objetivo
*Practicar agrupación lógica con fieldset y legend , además de controles de opción.*
**Enunciado**
*Crea una página titulada “Reserva de plaza en el bootcamp”. El formulario debe recoger esta información:*

### Grupo 1. Datos personales
- nombre completo;
- correo electrónico.

## Grupo 2. Modalidad
- La persona usuaria debe poder elegir una sola modalidad entre varias opciones.
## Grupo 3. Áreas de interés
- La persona usuaria debe poder marcar varias áreas de interés.
## Grupo 4. Motivación
- Debe existir un campo donde explicar brevemente por qué quiere participar.
- Lo que debes decidir tú
 Debes decidir:
- cuántas opciones ofreces en modalidad;
- cuántas opciones ofreces en áreas de interés;
- qué controles son más adecuados en cada grupo;
- qué campo necesita una ayuda adicional y cómo asociarla.

## Requisitos obligatorios
- debes usar al menos 2 fieldset ;
- cada fieldset debe tener su legend ;
- debes usar controles de elección única y controles de elección múltiple;
- debes usar aria-describedby al menos una vez;
- debe haber un botón final de envío.
