## Ejercicio 2 · Formulario de creación de cuenta
### Objetivo
*Practicar validación nativa y elección de controles adecuados.*
*Crea una página titulada “Crear cuenta de alumno”.*
*Dentro de main , construye un formulario que recoja esta información:*
 - nombre de usuario;
 - correo electrónico;
 - contraseña;
 - edad;
 - enlace a portfolio o perfil profesional;
 - aceptación de condiciones.
