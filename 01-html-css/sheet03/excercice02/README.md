## Ejercicio 2 · CSS en bloque <style>
### Objetivo 
**Objetivo**
*Practicar CSS dentro del propio documento HTML con la etiqueta Style*

**Se considera completado si:**
- El bloque <style> está correctamente colocado;
- Ee usan selectores básicos con sentido;
- Y ya se percibe una pequeña diferencia entre selector de etiqueta y selector de clase.
