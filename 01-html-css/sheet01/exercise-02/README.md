# Ejercicio 2: Pagina de rutas o lugares recomendados

## Descripcion
Creacion de una pagina web informativa que destaca tres lugares de interes dentro de una categoria especifica, aplicando buenas practicas de accesibilidad y semantica multimedia.

## Objetivo
Practicar la integracion de elementos multimedia mediante el uso de figuras, mejorar la accesibilidad con textos alternativos y estructurar informacion variada mediante listas y enlaces funcionales.

## Contenido de la ficha
* Seleccion de tres lugares o puntos de interes recomendados.
* Uso de imagenes descriptivas para cada localizacion.
* Listado de caracteristicas y procesos relacionados.
* Seccion de contacto con integracion de protocolos directos.

## Requisitos Tecnicos
* Estructura semantica avanzada (main, section, article, figure, figcaption).
* Implementacion de imagenes con atributos "alt" coherentes para accesibilidad.
* Combinacion de listas ordenadas (ol) y no ordenadas (ul).
* Uso de protocolos de comunicacion (tel: o mailto:) y enlaces externos seguros.
