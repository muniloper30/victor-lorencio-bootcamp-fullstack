# Ejercicio 1: Ficha de servicio profesional

## Descripcion
Creacion de una pagina web semantica que presenta un servicio profesional de creacion y diseño de portfolios para desarrolladores.

## Objetivo
Practicar la estructura basica de HTML5, el uso de etiquetas semanticas, metadatos y organizacion de contenido sin utilizar estilos CSS.

## Contenido de la ficha
* Presentacion del servicio de diseño.
* Listado de ventajas y caracteristicas.
* Pasos del proceso de trabajo.
* Seccion de contacto y enlaces externos.

## Requisitos Tecnicos
* Uso de etiquetas semanticas (header, section, footer, etc.).
* Enlaces externos con atributo target="_blank" y rel="noopener noreferrer".
* Listas ordenadas e iniciadas para el proceso y caracteristicas.
* Estructura de encabezados jerarquizada (h1, h2, h3).
