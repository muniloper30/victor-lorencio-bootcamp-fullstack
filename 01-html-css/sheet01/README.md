# Ejercicio 4: Artículo breve de divulgación técnica 

## Descripcion
Redacta una página con un artículo breve sobre uno de estos temas:
 - Por qué conviene escribir HTML semántico;
 - Errores habituales al empezar en desarrollo web;
 - Por qué la accesibilidad no es opcional;
 - Cómo organizar el contenido de una página antes de aplicar CSS;
 - Por qué el HTML sigue importando aunque uses frameworks.
## Objetivo
Practicar contenido textual con estructura más rica: citas, fragmentos de código y referencias.

## La página debe incluir 

* título principal;
* introducción;
* varios apartados;
* al menos un blockquote ;
* al menos un q ;
* al menos un cite ;
* al menos un code ;
* al menos un bloque pre + code ;
* un enlace externo a documentación o recurso relacionado.


## Condiciones obligatorias
 * En el enlace externo debes usar:
 - target="_blank"
 - rel="noopener noreferrer"
