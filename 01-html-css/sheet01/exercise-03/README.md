# Ejercicio 3: Página de preguntas frecuentes (FAQ)

## Descripcion
Creación de una página de soporte y ayuda informativa que resuelve dudas comunes sobre un servicio o herramienta, implementando un sistema de navegación interna mediante anclas.

## Objetivo
Practicar la jerarquía de contenidos, la navegación fluida dentro del mismo documento y la organización lógica de secciones para mejorar la experiencia de usuario y la accesibilidad.

## Contenido de la ficha
* Cabecera con título principal y una breve introducción al servicio.
* Menú de navegación interna con enlaces directos a las distintas categorías.
* Bloques de preguntas frecuentes (mínimo cuatro) agrupados por temática.
* Sección final de contacto o soporte técnico para resolución de dudas adicionales.

## Requisitos Tecnicos
* Estructura semántica obligatoria (`header`, `nav`, `main`, `section`).
* Implementación de navegación mediante anclas internas (`href="#..."` e `id`).
* Jerarquía de encabezados (H1-H3) coherente y lógica según el contenido.
* Uso de textos descriptivos en los enlaces para asegurar que la página sea fácil de recorrer.
